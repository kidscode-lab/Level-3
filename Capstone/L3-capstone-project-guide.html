How the AI Integration Works:

# The brain module handles AI communication for you: import google.generativeai as genai # Configuration (already done in __init__) genai.configure(api_key=your_api_key) self.model = genai.GenerativeModel('gemini-2.0-flash-lite') # Your job: Create effective prompts and call: response = self._get_ai_response(your_prompt)

AI Prompting Best Practices

  • Be specific about the audience (middle school students)
  • Request plain text output to avoid formatting issues
  • Give clear structure instructions (bullet points, answer keys, etc.)
  • Include examples of the desired output format when helpful
  • Keep prompts concise but comprehensive

‚ö†Ô∏è Common Mistakes to Avoid

Learn from these frequent student errors

Prompt Engineering Issues

  • Mistake: Vague prompts that don't specify output format
  • Result: Inconsistent AI responses with unwanted formatting
  • Fix: Always specify "plain text only" and exact format requirements

History Logging Problems

  • Mistake: Forgetting to call _append_user() and _append_bot()
  • Result: Chat history doesn't save interactions properly
  • Fix: Always log both the user request and AI response at the end of each method

API Response Handling

  • Mistake: Not checking if the AI response contains expected content
  • Result: Empty or malformed responses break the application
  • Fix: The _get_ai_response() method already handles errors, just use the returned text

Text Length Issues

  • Mistake: Sending very long text to the AI without truncation
  • Result: API token limits exceeded, causing errors
  • Fix: Use [:4000] to limit text length in summarize_text()
